## Kesimpulan Pembelajaran (Tugas 4, 5, & 6)

### Tugas 4: Struktur Data Python
Pada tugas ini, saya mempelajari cara mengelola koleksi data menggunakan **List, Tuple, Set, dan Dictionary**. Pemahaman ini sangat krusial untuk menyimpan informasi secara terstruktur, seperti daftar fitur dalam proyek aplikasi **KERIS**.

### Tugas 5: Fungsi dan Pemrograman Berorientasi Objek (OOP)
Tugas ini fokus pada pembuatan kode yang dapat digunakan kembali (*reusable*) melalui **Function** dan pengelompokan data serta logika melalui **Class**. Konsep ini membantu dalam membangun sistem yang lebih kompleks, seperti manajemen data mahasiswa.

### Tugas 6: Library NumPy, Pandas, dan File I/O
Tugas terakhir ini memperkenalkan alat untuk analisis data. **NumPy** mempermudah perhitungan statistik numerik, sementara **Pandas** sangat membantu dalam mengolah data berbentuk tabel (DataFrame). Saya juga mempelajari cara mengekspor hasil analisis ke dalam file teks (.txt) secara otomatis.

Secara keseluruhan, rangkaian tugas ini memberikan fondasi yang kuat bagi saya sebagai mahasiswi Teknik Informatika UISU untuk melangkah ke tahap pengembangan AI yang lebih lanjut.
